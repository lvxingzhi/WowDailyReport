# 大秘境分数行情日报 - 开发文档

## 1. 项目概述

### 1.1 目标
为《魔兽世界》大秘境日报视频内容创作，提供一套可截图嵌入视频的数据信息图页面。

### 1.2 使用场景
内容创作者在视频制作过程中，通过截图方式将各信息模块嵌入视频对应位置。每屏独立截图，无需滚动。

### 1.3 数据来源
- 原始数据：`大秘境分数行情_至暗之夜S1.xlsx`（包含「每日数据」「国家队阵容」两个 Sheet）
- 解析方式：Python + openpyxl
- 输出格式：JSON

## 5. 幻灯片设计（共 7 屏）

### 5.1 幻灯片列表

| 序号 | 名称 | 路由 | 展示内容 |
|------|------|------|---------|
| 1 | 赛季概览 | #slide-1 | 赛季名、当前周/剩余周、最新数据日期 |
| 2 | 分数行情 | #slide-2 | 第一名/1%/0.1% 分数历史趋势折线图 |
| 3 | 第一名高光 | #slide-3 | 第一名角色名、职业、分数、队伍成员大卡片 |
| 4 | 国家队生态 | #slide-4 | 国家队占比环形图 + 非国家队占比数字 |
| 5 | 信仰专精 | #slide-5 | 3个冷门专精及其最高分卡片（横向排列） |
| 6 | 副本限时 | #slide-6 | 8个副本限时层数柱状图 |
| 7 | 铁钥匙 | #slide-7 | 24铁/23铁人数趋势折线图 |

### 5.2 交互设计
- 键盘 ← → 键切换幻灯片
- 点击左右箭头按钮切换
- 底部圆点指示器（点击可跳转）
- URL hash 定位（支持直接分享某个幻灯片）
- 每屏固定 16:9 比例，内容垂直水平居中
- 安全边距：上下 5%，左右 8%（确保不被视频裁剪）

### 5.3 视觉规范

| 元素 | 规范 |
|------|------|
| 背景色 | 深色渐变 `#0a0e17` → `#141c2b` |
| 主强调色 | 金色 `#c9a84c` |
| 次强调色 | 橙色 `#d4783b` |
| 数据大字 | 白色 `#ffffff`，字号 48-72px |
| 辅助文字 | 灰色 `#8892a4`，字号 18-24px |
| 卡片背景 | 半透明 `rgba(255,255,255,0.05)` + 边框 `rgba(201,168,76,0.2)` |
| 图表主题 | ECharts 暗色主题 |
| 字体 | 系统无衬线字体 |
